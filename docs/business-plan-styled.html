1. エグゼクティブサマリー

GORU GYMは、「筋トレ × ゴリラ育成 × 継続ゲーム」を融合した初心者向けトレーニング継続支援アプリである。PWAとして開発されており、ログイン不要でも即時利用できる軽量性と、ゲーム感覚で継続を促す設計を強みとする。現在は https://gori-gym.vercel.app で公開済みであり、筋トレ記録、ゴリラ育成、AI応答、ランキング、フレンド、ギルドなど、継続支援の基盤機能はすでに実装済みである。

本事業が解決するのは、ジム初心者の極めて高い離脱率である。多くの初心者は入会後3日〜2週間で通わなくなる傾向があり、その背景には「何をすればよいかわからない」「フォームが不安」「成果実感まで時間がかかる」「ジム空間への心理的ハードルが高い」といった構造的課題がある。

GORU GYMは、Duolingoの継続設計思想を筋トレ領域に応用し、「毎日やる」ではなく「週4回続ける」ことを最適化する。記録管理ではなく行動変容を目的とし、ゴリラ育成や非同期対戦、ギルド対抗戦などのエンタメ要素を通じて、初心者が成果実感を得る前に離脱してしまう谷を越えさせることを目指す。

収益モデルは、フリーミアム課金、トレーナーマッチング手数料、企業ジム向けB2B導入の3本柱を想定する。

2. 課題と市場機会

2.1 解決すべき課題

ジム市場は拡大している一方で、初心者の継続率には大きな課題がある。特に初心者は以下の理由で離脱しやすい。

  • 何の種目から始めればよいかわからない
  • マシンの使い方やフォームに不安がある
  • 体型変化や筋力向上の成果が見えるまで時間がかかる
  • 周囲が経験者に見えて心理的に萎縮する
  • 記録アプリを入れても入力負荷が高く、習慣化前に面倒になる

2.2 市場背景

日本のフィットネス市場は、24時間ジムの普及、健康経営需要、ボディメイク文化の浸透を背景に拡大基調にある。国内フィットネスクラブ関連市場は数千億円規模と推定される。

3. プロダクト概要

3.1 コンセプト

「Duolingoの筋トレ版」。毎日実施を強制するのではなく、週4回の達成を最適化する。ユーザーはトレーニングを行うことでゴリラを育成し、継続そのものが楽しくなる体験を得る。

3.2 実装済み機能（Ver1〜2）

  • 筋トレ記録 / ゴリラ育成（Lv5、毎月リセット） / AIゴリラ応答
  • PWA対応 / スプラッシュ画面 / データバックアップ
  • Firebase認証（Google / メール） / Firestoreデータ同期
  • フレンド機能（リクエスト / 一覧 / ゴリラ閲覧 / いいね）
  • ランキング（月間総重量 / 日数、全体 / フレンド内）
  • ギルド機能（5人チーム、統計、チャット）

5. ターゲットユーザーペルソナ

ペルソナA：24歳 男性 会社員

24時間ジムに入会したばかり。何をすればよいかわからず、周囲の視線が気になる。ゲーム好きでSNS利用頻度が高い。

ペルソナB：29歳 女性 事務職

運動不足解消が目的。器具に不慣れでフォームが不安。短時間で達成感を得たい。

ペルソナC：21歳 学生

友人と一緒に筋トレを続けたい。共有・競争・ネタ性に反応しやすい。

最重要KPIは記録数ではなく、週次継続率・4週間継続率・8週間継続率。

12. まとめ・ビジョン

GORU GYMの本質は、筋トレ記録アプリではなく、「運動が続かない人を、続けられる人に変える仕組み」である。

すでに公開済みのPWAとして実装基盤を持ち、ソーシャル機能や育成要素も備えているため、今後はDuolingo式の継続設計を完成させることで、初心者向けフィットネス領域における独自ポジションを確立できる。

将来的には、初心者向け継続支援アプリから、AIトレーナー、食事分析、法人向け健康経営支援までを包含する総合フィットネス習慣化プラットフォームへ進化することを目指す。
