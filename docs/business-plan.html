GORU GYM 事業企画書

1. エグゼクティブサマリー

GORU GYMは、「筋トレ × ゴリラ育成 × 継続ゲーム」を融合した初心者向けトレーニング継続支援アプリである。PWAとして開発されており、ログイン不要でも即時利用できる軽量性と、ゲーム感覚で継続を促す設計を強みとする。現在は https://gori-gym.vercel.app で公開済みであり、筋トレ記録、ゴリラ育成、AI応答、ランキング、フレンド、ギルドなど、継続支援の基盤機能はすでに実装済みである。

本事業が解決するのは、ジム初心者の極めて高い離脱率である。多くの初心者は入会後3日〜2週間で通わなくなる傾向があり、その背景には「何をすればよいかわからない」「フォームが不安」「成果実感まで時間がかかる」「ジム空間への心理的ハードルが高い」といった構造的課題がある。既存の筋トレアプリの多くは中上級者向けの記録・分析ツールであり、初心者の最重要課題である“継続”そのものをプロダクトの中心に据えたサービスは限定的である。

GORU GYMは、Duolingoの継続設計思想を筋トレ領域に応用し、「毎日やる」ではなく「週4回続ける」ことを最適化する。記録管理ではなく行動変容を目的とし、ゴリラ育成や非同期対戦、ギルド対抗戦などのエンタメ要素を通じて、初心者が成果実感を得る前に離脱してしまう谷を越えさせることを目指す。

収益モデルは、フリーミアム課金、トレーナーマッチング手数料、企業ジム向けB2B導入の3本柱を想定する。まずはB2Cで継続率改善に特化したプロダクトとしてユーザー基盤を形成し、その後にAIトレーナー、食事分析、法人向け福利厚生プランへ拡張することで、LTVの高いヘルスケア・フィットネスプラットフォームへ進化させる。

2. 課題と市場機会

2.1 解決すべき課題

ジム市場は拡大している一方で、初心者の継続率には大きな課題がある。一般的にフィットネスクラブや24時間ジムでは、新規入会者の一定割合が短期間で来館頻度を落とし、実質的に休眠化する。特に初心者は以下の理由で離脱しやすい。

  • 何の種目から始めればよいかわからない
  • マシンの使い方やフォームに不安がある
  • 体型変化や筋力向上の成果が見えるまで時間がかかる
  • 周囲が経験者に見えて心理的に萎縮する
  • 記録アプリを入れても入力負荷が高く、習慣化前に面倒になる

この課題は、単なる「記録のしやすさ」ではなく、「初心者が最初の数週間を乗り切るための心理設計・行動設計」の不足に起因している。

2.2 市場背景

日本のフィットネス市場は、24時間ジムの普及、健康経営需要、ボディメイク文化の浸透を背景に拡大基調にある。国内フィットネスクラブ関連市場は数千億円規模と推定され、加えてホームトレーニング、オンライン指導、サプリメント、ヘルスケアアプリを含む周辺市場まで広げると、デジタルフィットネス領域はさらに大きい。

スマートフォンアプリ市場においても、運動記録・食事管理・ヘルスケア支援アプリは継続的な需要があり、特に若年層〜30代の初心者層では「難しい分析」よりも「続けられる仕組み」へのニーズが高い。

2.3 TAM / SAM / SOM

以下は日本市場を前提とした妥当な推定値である。

TAM（Total Addressable Market）

  • 国内フィットネス参加人口を約500万人規模と仮定
  • うちスマホアプリ活用に親和性の高い層を70%と仮定
  • 年間課金可能額を1人あたり平均6,000円と仮定

TAM = 約210億円/年

計算式：500万人 × 70% × 6,000円 = 210億円

SAM（Serviceable Available Market）

  • 主対象を「ジム入会から1年未満の初心者」に限定
  • 初心者比率を全体の20%と仮定
  • そのうちアプリ利用意向のある層を60%と仮定

SAM = 約36億円/年

計算式：500万人 × 20% × 60% × 6,000円 = 36億円

SOM（Serviceable Obtainable Market）

  • 初期3〜5年で獲得可能な国内ニッチシェアをSAMの1〜3%と想定
  • まずはSNS拡散と口コミで初心者コミュニティを形成

SOM = 約0.36億〜1.08億円/年

計算式：36億円 × 1〜3%

このSOMは、単体アプリ課金のみを前提とした保守的な試算であり、将来的なB2B導入やトレーナーマッチング収益を含めると上振れ余地がある。

2.4 市場機会の本質

GORU GYMの市場機会は、「筋トレ市場」そのものよりも、「初心者が継続できずに脱落している巨大な未充足領域」にある。既存サービスが中上級者の効率化を支援しているのに対し、GORU GYMは初心者の継続率改善という未解決課題に集中することで、明確なポジショニングを築ける。

3. プロダクト概要

3.1 プロダクトコンセプト

GORU GYMは、筋トレ初心者が「続けられる」ことを最優先に設計されたゲーム型フィットネスアプリである。コンセプトは「Duolingoの筋トレ版」であり、毎日実施を強制するのではなく、週4回の達成を最適化する。ユーザーはトレーニングを行うことでゴリラを育成し、継続そのものが楽しくなる体験を得る。

3.2 現在の実装済み機能（Ver1〜2）

  • 筋トレ記録機能
  • ゴリラ育成機能（Lv5、毎月リセット）
  • AIゴリラ応答
  • PWA対応
  • スプラッシュ画面
  • データバックアップ
  • Firebase認証（Google / メール）
  • Firestoreデータ同期
  • フレンド機能（リクエスト / 一覧 / ゴリラ閲覧 / いいね）
  • ランキング（月間総重量 / 日数、全体 / フレンド内）
  • ギルド機能（5人チーム、統計、チャット）

これにより、単なる個人記録アプリではなく、ソーシャル継続基盤まで備えた初期プロダクトがすでに成立している。

3.3 技術基盤

  • PWA
  • Vite
  • React
  • TypeScript
  • Firebase Authentication
  • Firestore
  • オフラインファースト設計

PWAであることにより、アプリストア審査なしで高速改善が可能であり、導入障壁も低い。ログイン不要で即利用できる点は、初心者の初回離脱を防ぐ重要なUX上の優位性である。

3.4 ロードマップ

Ver3：Duolingo式継続設計の完成

  1. 今日のミッションを1つだけ提示
  2. 週間ストリーク + XP導入（週4達成で継続判定）
  3. ゴリラ反応強化（サボると退化・悲しむ、トレすると褒める）
  4. レベルアップ演出強化
  5. ゴリラ対決（非同期PvP、トレ実績をステータス変換、自動バトル演出）
  6. ギルド対抗戦

Ver4：初心者ガイド機能

  • マシン説明
  • 重量自動提案
  • フォームガイド

Ver5：AIトレーナー + 収益化

  • オンライントレーナー連携
  • プレミアムプラン
  • 広告導入

Ver6：食事・体重管理拡張

  • AI食事分析（写真からカロリー推定）
  • 体重管理グラフ

3.5 提供価値

GORU GYMが提供する価値は以下の3点に集約される。

  1. 継続支援：初心者が最も挫折しやすい最初の数週間を乗り越えやすくする
  2. 心理的ハードル低減：ゴリラ育成やミッション提示により、ジムへの恐怖感を軽減する
  3. 成果実感の前倒し：身体変化が出る前でも、ゲーム内成長や仲間との交流で達成感を得られる

4. 競合分析

4.1 競合環境

フィットネスアプリ市場には、ワークアウト記録、動画レッスン、食事管理、総合ヘルスケアなど複数カテゴリが存在する。GORU GYMの直接競合は「筋トレ記録アプリ」および「初心者向け運動支援アプリ」だが、差別化軸は“記録”ではなく“継続ゲーム化”にある。

6. ビジネスモデル・収益計画

6.1 収益モデル

1. フリーミアム

  • 基本機能は無料
  • プレミアム機能：
    • 広告非表示
    • 追加ゴリラスキン
    • 詳細分析
    • 高度なAIアドバイス

想定価格帯：月額480円〜980円

2. トレーナーマッチング手数料

  • 初心者とオンライン/提携トレーナーを接続
  • セッション予約や継続プラン販売時に手数料を徴収

想定手数料率：15〜25%

3. B2B導入

  • 企業ジム、福利厚生、健康経営施策向けに提供
  • 法人単位で利用ライセンス契約
  • 継続率改善や運動習慣化をKPIとして提案可能

想定価格帯：月額5万円〜30万円/法人

6.2 収益化の順序

  1. B2C無料でユーザー獲得
  2. プレミアム導入でARPU向上
  3. AIトレーナー・マッチングで高単価収益化
  4. B2B展開で安定収益化

6.3 主要KPI

  • 月間アクティブユーザー数（MAU）
  • 新規登録/初回利用者数
  • 1週継続率
  • 4週継続率
  • 週4達成率
  • プレミアム転換率
  • CAC（顧客獲得単価）
  • LTV（顧客生涯価値）

6.4 単位経済性の想定

初期の保守的な想定として、

  • プレミアム転換率：3〜5%
  • プレミアム月額：680円
  • 広告ARPU：無料会員あたり月額30〜80円
  • トレーナーマッチング利用率：MAUの1〜2%

とすると、ユーザー基盤拡大後は無料会員の広告収益と一部課金ユーザーのサブスク収益を組み合わせたハイブリッドモデルが成立する。

7. マーケティング戦略

7.1 基本戦略

GORU GYMは、初心者の共感を起点としたSNS拡散と、ゲーム性によるUGC創出を主軸に成長させる。広告大量投下よりも、「筋トレ初心者あるある」と「ゴリラ育成の面白さ」を掛け合わせた自然流入を重視する。

7.2 獲得チャネル

  • X（旧Twitter）
  • TikTok
  • Instagram Reels
  • YouTube Shorts
  • 筋トレ初心者向けブログ/SEO
  • ジム系インフルエンサーとの提携
  • 学生コミュニティ・新社会人コミュニティ

7.3 SNS拡散設計

GORU GYMは、プロダクト自体に拡散導線を組み込める。

  • 育成したゴリラの見た目変化をシェア
  • 週間ストリーク達成画像をシェア
  • ギルド順位や対戦結果をシェア
  • 「今日のゴリラの一言」をSNS投稿可能にする
  • 月末リセット時の成長結果をカード化して共有

これにより、単なる成果報告ではなく、キャラクターコンテンツとして拡散されやすい。

7.4 初期グロース仮説

  • フェーズ1：個人開発ストーリー発信で初期ファン獲得
  • フェーズ2：筋トレ初心者コミュニティで口コミ拡大
  • フェーズ3：紹介施策・ギルド招待でネットワーク効果を形成
  • フェーズ4：継続率改善データを武器に法人提案へ展開

7.5 ブランドメッセージ

  • 「筋トレが続かない人のためのアプリ」
  • 「記録するより、まず続ける」
  • 「あなたの筋トレを、ゴリラが見守る」

9.1 現在の体制

現時点では、個人開発者1名による開発・企画体制で推進している。

  • 役割：プロダクト企画
  • 役割：UI/UX設計
  • 役割：フロントエンド開発
  • 役割：バックエンド/インフラ実装
  • 役割：ユーザー検証

個人開発であることは意思決定の速さと改善速度において優位性がある一方、開発リソースには制約がある。そのため、優先順位は「継続率改善に直結する機能」に集中させる。

12. まとめ・ビジョン

GORU GYMは、筋トレ初心者が最もつまずきやすい「最初の継続」をゲームの力で支えるプロダクトである。既存の筋トレアプリが記録や分析を中心に進化してきたのに対し、GORU GYMは“続けること”を主目的に据えることで、明確な差別化を実現している。

すでに公開済みのPWAとして実装基盤を持ち、ソーシャル機能や育成要素も備えているため、今後はDuolingo式の継続設計を完成させることで、初心者向けフィットネス領域における独自ポジションを確立できる可能性が高い。

将来的には、初心者向け継続支援アプリから、AIトレーナー、食事分析、法人向け健康経営支援までを包含する総合フィットネス習慣化プラットフォームへ進化することを目指す。GORU GYMの本質は、筋トレ記録アプリではなく、「運動が続かない人を、続けられる人に変える仕組み」である。
